## Shopping platform

Made with:
	*Backend: ASP.NET CORE WebAPI 3.1.8  
	*Frontend:angular v9

#### Application allows:
* adding/removing items for sell
* buying items
* creating shopping card
* browsing transaction history
	
	
## To run this application...

##### you need to run:
* API: ``` dotnet run ```
* Angular app:  ```ng serve```
